# JsonParser

A java version of [](https://github.com/miloyip/json-tutorial)

## 文件目录

```
Parser.java                // json解析器
ParserResult.java          // 解析状态 （如OK, INVALID_VALUE等）
Value.java                 // 解析树的节点
ParserType.java            // 节点类型
ParserTest.java            // 单元测试类
TestException.java         // 测试类抛出的异常
Generator.java             // json生成器
```
